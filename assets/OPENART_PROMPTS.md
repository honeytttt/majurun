## HOW TO GENERATE

1. Go to openart.ai → Image (for static) or Video/Seedance (for GIFs/animations)
2. Paste prompt, select model, set aspect ratio
3. Download and save to the matching folder below
4. Run upload script to push to Cloudinary (see CLOUDINARY_UPLOAD.md)

### Folder structure to save into:
